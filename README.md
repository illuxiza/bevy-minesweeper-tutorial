# Bevy 扫雷游戏开发教程

本教程用于介绍如何使用[Bevy(0.11.3)](https://github.com/bevyengine/bevy)开发一个类似windows扫雷游戏。
希望你能在这里入门Bevy并享受使用它开发游戏的乐趣。


主要目标：
- [ ] 学习Bevy的基础功能和ECS系统
- [ ] 学习如何使用工具调试
- [ ] 开发一个Bevy Plugin
- [ ] 其他待添加。。。
